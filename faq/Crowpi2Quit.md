# How to quit the Crowpi2 software?

Run the command `killall Crowpi2` in a terminal.  
1. Press **Ctrl**+**Alt**+**T**, a new **Terminal** window appears.  
2. At the command prompt, type `killall Crowpi2` and press **Enter**.
